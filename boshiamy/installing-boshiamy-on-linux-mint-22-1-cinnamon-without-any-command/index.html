[無需指令] 在 Linux Mint 22.1 Cinnamon 上安裝嘸蝦米輸入法

發表於 2025-01-30 分類於 boshiamy
文章字數： 1.9k 所需閱讀時間 ≈ 3 分鐘

前言

本篇文章，主要是分享在 Linux Mint 22.1 上，不需要任何指令，就能夠安裝及設定好嘸蝦米輸入法的過程。

安裝過程

  1. 開啟「軟體管理員」。

  2. 在「軟體管理員」裡，將 fcitx5-table-boshiamy 找出來，再直接點按【fcitx5-table-boshiamy】進入下一個步驟。

  3. 點「軟體管理員」介面右上角的【安裝】按鈕，此時，「軟體管理員」會跳出一個 會安裝相關軟體 的說明，按【繼續】往下走。

  4. 安裝 fcitx5-table-boshiamy 前，系統會有一個身份核對的對話框，只要將密碼輸入即可。另外，眼尖的伙伴可能會發現「軟體管理員」介面右上角在跑安裝進度條。

  5. 安裝完成後，可以看到「軟體管理員」介面右上角的【安裝】按鈕，變成【移除】按鈕，這就表示已經安裝完成（有點可惜，如果能夠跳出訊息視窗說明「已完成安裝」的話，就更棒）。接著，要重新啟動系統。

設定過程

  1. 重新進入系統後，工作列上會出現一個【小鍵盤】圖示。右鍵點一下【小鍵盤】，從跳出的功能表可以看到「嘸蝦米」還沒有掛載進來。

  2. 點一下功能表裡的【設定】，進入「Fcitx 設定」介面。

  3. 「Fcitx 設定」介面中，右手邊「可用輸入法：」裡，排位第 1 的就是「嘸蝦米」。

  4. 我們要做的就是記得將「嘸蝦米」加到左手邊的「目前輸入法：」項目裡。

  5. 按下【確定】、【套用】按鈕後，以右鍵點按工作列上的【小鍵盤】圖示，就會發現「嘸蝦米」進入功能表內。

  6. 開一個軟體來試試「嘸蝦米」（這裡是用 LibreOffice Writer）。

後記

  1. 本次，我所選擇的套件是 fcitx5-table-boshiamy，而不是 fcitx5-table-extra。

    fcitx5-table-extra 涵蓋範圍比 fcitx5-table-boshiamy 還要廣。

    fcitx5-table-extra provides extra table for Fcitx5, including Boshiamy, Zhengma, Cangjie, and Quick.

    以往，我選用 Fcitx5 的時候，安裝嘸蝦米時，就是搭 fcitx5-table-extra 這支套件。然而，我最近注意到 Debian 與 ubuntu 及其衍生發行版，單獨將「嘸蝦米」打包成 fcitx5-table-boshiamy 這支套件，並收錄至軟體庫。也就是說，只想安裝「嘸蝦米」的伙伴，可以選擇安裝 fcitx5-table-boshiamy 這支套件即可。

  2. 在 DistroWatch 上，Linux Mint 霸榜「網頁參觀排名」好一段時間，就我自己的使用經驗，推測應該是 Linux Mint 解決不少 End-User 會碰到的困擾。

    就以這次分享的 22.1 版（Xia）來說，官方在「New Features in Linux Mint 22.1 ‘Xia’ - Linux Mint」說明裡，一開始就談到「Modernization of APT dependencies」這一項更新。

    這項更新，在我今天以安裝嘸蝦米的過程來做實測，我認為非常符合我個人對 Linux 的期待，如果其他 App 的安裝，也如同本次的嘸蝦米輸入法安裝這般簡單，我相信，可能會讓更多想要使用 Linux，卻因為看到需要下指令就望之卻步的人，有意願嘗試看看。換句話說，正要入門 Linux 的伙伴，選這一支 Distro 做為起手式，應該會多一點愉快感，尤其對習慣 Windows 或 macOS 的伙伴而言，使用 Linux Mint 的桌面系統，應該比較能夠無縫接軌。

  3. 本次安裝使用的 ISO 檔是 linuxmint-22.1-cinnamon-64bit.iso。
