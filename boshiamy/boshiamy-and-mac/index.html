嘸蝦米與 Mac

發表於 2008-01-18 分類於 boshiamy
文章字數： 1k 所需閱讀時間 ≈ 2 分鐘

隨著嘸蝦米 Mac 版的演進，我已經不打算再寫一次安裝的方式。

日前分享的原因，主要是我注意到 Mac OS X 的誕生帶來很多 Switch Users ，有很多第一次接觸 Mac 的使用者對於蘋果了解有限，而國內的資源也有限，況且行易公司在 Mac OS X 上釋出的第二個重要版本時安裝說明是以英文撰寫，於是我起草安裝過程，把幾個重要關鍵步驟，包括從第一版升級到第二版需要注意的事項紀錄下來，文帶圖的說明方式供大家參考。

後來，嘸蝦米進入第三個重要版本的發展，其實只要滑鼠左鍵快點兩下，一步接著一步就可以完成安裝，貼出安裝方法已顯得多餘。惟，之前發完文章後，陸陸續續有伙伴跟我索取嘸蝦米安裝檔，一方面遭受到我的捥拒，另一方面我建議伙伴們不妨直接去購買正式授權的軟體。

如今再談嘸蝦米時，有股強烈的念頭跟大伙分享一則小故事：事情是在我就讀大學時發生的，當時候有一顆很可愛的蘋果： iMac 誕生，然而只用 USB 1.1，對於原本身穿昂貴 SCSI 的老 Mac 而言，簡直就是離經叛道，更帥氣的是這個毛頭小伙子竟然把 Floppy 讀取機從身上拔除，造成極大的不便，對於已經學會並慣用嘸蝦米的我，買一部 iMac Rev.B 已經花掉身上的積蓄。若要使用嘸蝦米，除了 1,600 元的正版軟體授權費，還得再花錢買部外接的 Floppy 讀取機，最後我決定騎著車，載著心愛的女友（如今是內人），親自跑一趟行易公司，請行易公司幫個忙，後來行易公司表示，我可以拿正版的 Floppy 跟行易公司換 CD-Rom ，該 CD-Rom 是行易公司當下燒錄給我的，從此深刻了解使用正式授權軟體的好處，也更樂意少喝一瓶飲料，存下來必要時買好用的軟體授權。附帶一提的是聖堂 FTP 有提供嘸蝦米 1.1 Demo 版（說明檔與載點），我在大學的時代（約十年前）跟行易公司聯繫，行易公司的開發、客服人員都不曉得，很可能是這一支 Demo 年代久遠，已不可考。

Mac OS 9 走入 Classical 時代， Mac OS X 正式隨機搭售之初，蝦米族是透過好心人將 Table 轉換後以相當不便的 Return 鍵入字，可謂為黑暗時期。後來，行易公司釋出 Mac OS X 第一版，蝦米族們拾回 Space 鍵入字的快感，這才又走向光明。隨著第二版、第三版的出現，顯然行易公司確實還有支援 Mac 的蝦米族，因此我仍然建議蝦米族們：請支持正版！
