安裝 Fcitx5 for Android(小企鵝輸入法5)與嘸蝦米(2025 年 3 月 18 日更新)

發表於 2024-04-28 分類於 boshiamy
文章字數： 1.9k 所需閱讀時間 ≈ 3 分鐘

前言

老早以前，我就注意到 Fcitx5 for Android 這支 App。

雖然，我曾經抱著好奇的心態下載與使用 Fcitx5 for Android 這支 App，但是，可能是我已經有官方的嘸蝦米輸入法 App，所以，啟動 Fcitx5 for Android 後，並沒有仔細研究如何把嘸蝦米輸入法表格掛載上去。一個機緣之下，發現已經有網友成功處理完成，我參考網友的說明後，發現設定的要點與程序，真的是「藏得有點深」，所以，決定以螢幕截圖的方式，紀錄自己走過的步驟。

安裝過程

安裝 Fcitx5 For Android 這支主程式

  1. 對安裝 App 已經非常熟悉的伙伴，或可略過，直接去參考「2.2. 掛載嘸蝦米輸入法」這個段落。

  2. 先跑一趟 Google Play 應用程式商店（商店傳送門：Fcitx5 for Android - Google Play 應用程式），把 Fcitx5 For Android 安裝進來。

  3. 安裝完成後，可以直接從「Play 商店」按【開啟】。

  4. 初次啟動，需要走幾個步驟。首先，就是先點【啟用輸入法】。

  5. 點按【啟用輸入法】後，會帶到下圖所示之介面，我們要做的是把「小企鵝輸入法」右手邊的開關點開。

  6. 打開開關後，「小企鵝輸入法」會帶出一個「注意」的說明視窗，有興趣的伙伴，可以讀一下，讀完記得點【確定】。

  7. 開啟完「小企鵝輸入法」，會回到下圖所示之介面，我會直接再點一下【小企鵝輸入法】，直接進入程式的設定畫面。

  8. 我選擇點下【授予權限】。

  9. 我會點【允許】，同時停留在這個介面上，以便接下來掛載嘸蝦米輸入法。※如果因故跳開這個介面，也不用擔心，只要找到並點下「小企鵝輸入法」的主程式圖示，還是可以回到熟悉的畫面。

掛載嘸蝦米輸入法

  1. 到掛載嘸蝦米輸入法表格相關檔案的設定前，要走不少步驟。第一步先點【附加元件】。

  2. 接著點「碼表」右手邊的【齒輪】圖示。

  3. 點按【管理碼表輸入法】。

  4. 點按右下角的【+】號的圓形圖示。

  5. 走到這裡，有 2 種方法可以將嘸蝦米輸入法表格相關檔案掛進來，我選用的方法是跑一趟 GitHub：fcitx5-table-extra/tables at master · fcitx/fcitx5-table-extra · GitHub ，把 boshiamy.conf.in 與 boshiamy.txt 載到行動裝置裡面，再分別掛載上去。

  6. 沒有問題的話，按下【確定】，就會看到「碼表輸入法」介面底下，多了一個「Boshiamy」。

  7. 點按左上角的【←】圖示 3 次，回到一開始的設定畫面，再從【輸入法】進去。

  8. 點右下角的【+】號的圓形圖示，把「Boshiamy」新增進來。

切換輸入法至「小企鵝輸入法」

  1. 點右下角的【地球鍵】，切換輸入法至「小企鵝輸入法」。

  2. 再點一下【English】左手邊的【地球鍵】。

  3. 看到【English】切換至【Boshiamy】，才算是真正切換到嘸蝦米輸入法。最後，再試著敲幾個字，看看是否安裝成功。

後記

  1. 本次安裝使用的硬體是 Pixel 7，系統版本是 Android 15；「小企鵝輸入法」（即 Fcitx5 for Android）的版本是 0.1.0-0-g78c03d12；boshiamy.conf.in 是 Commits on Apr 29, 2024；boshiamy.txt 是 Commits on Aug 25, 2019。

  2. Fcitx，可以說是我目前使用 Linux 時，9 成以上會選用的輸入法框架。想不到 Android 版本，也非常好用，鍵盤的設定跟我的習慣很對味。

  3. 走一趟 GitHub 下載表格檔，包括：「輸入法設定檔」與「碼表字典檔案」，我都是下載「Raw file content」。

  4. 下載的檔案中，boshiamy.conf.in（即「輸入法設定檔」），檔名會變成 boshiamy.conf.in.txt，我直接從行動裝置上進行重新命名的動作（下圖）。
