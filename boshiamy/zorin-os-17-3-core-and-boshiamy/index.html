Zorin OS 17.3 Core 與嘸蝦米

發表於 2025-04-04 分類於 boshiamy
文章字數： 601 所需閱讀時間 ≈ 1 分鐘

前言

Zorin OS 是一支以 ubuntu 為基礎的 Linux Distro。

安裝及使用嘸蝦米

  1. 在終端機(Terminal)下一道指令：sudo apt install fcitx-table-boshiamy，安裝輸入法框架 fcitx(4) 與嘸蝦米表格(fcitx-table-boshiamy)。

  2. 從【開始功能表】→【設定值】→【地區和語言】→【管理安裝的語言】，進入「語言支援」的設定介面。

  3. 在「語言支援」設定介面的下方，點按「鍵盤輸入法系統：」右手邊的下拉式選單，並選擇【Fcitx 4】。

  4. 下一道指令：sudo apt remove ibus*，移除 iBus 及其相關套件。我認為主要目的是不會再看到「zh」字樣，以我自己使用的經驗，不移除也不會影響 fcitx(4)。

  5. 重新啟動系統。下圖是透過終端機(Terminal)下一道指令：reboot，去重新啟動系統。

  6. 再次進入系統後，可以看到工作列上面多出一個小鍵盤，並且點進小鍵盤的功能選單，可以看到「嘸蝦米」的字樣，也就是說，嘸蝦米已經是預設且唯一的中文輸入法。

  7. 找個程式來測試一下。

後記

  1. 本次安裝使用的 ISO 檔版本是 Zorin-OS-17.3-Core-64-bit.iso。

  2. Zorin OS 17.3 這一版，立基於 ubuntu 22.04 LTS，所以，還沒有將 fcitx5-table-extra 納入軟體庫。
