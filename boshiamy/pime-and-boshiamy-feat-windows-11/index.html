PIME 輸入法與嘸蝦米 feat. Windows 11

發表於 2025-03-18 分類於 boshiamy
文章字數： 1.1k 所需閱讀時間 ≈ 2 分鐘

前言

來分享一下，我用 PIME 搭配嘸蝦米的吃蝦體驗。

安裝過程

  1. 下載之後，執行安裝程式。基本上沒有太大的難度，就是點【OK」與【我同意】。

  2. 走到這裡，由於我要搭配嘸蝦米，所以，必須勾選「蝦米（須自備正版表格檔）」，勾選完要按【安裝】。

  3. 接著會跳出一個訊息，跟我們說：「請選取 liu-uni.tab 檔案所在位置。」我所使用的 liu-uni.tab 是取自「嘸蝦米輸入法 J 標準版 (x64) rev. 469」，這一個版本的 liu-uni.tab 預設是放在「C:\Program Files\BoshiamyTIP」這個路徑底下。

  4. 選完記得按【開啟】。

  5. 通常，安裝程式就會自己走完安裝程序。

官方嘸蝦米與 PIME 的介面「微」比較

  1. 官方嘸蝦米輸入法的選字框是「直排」的。

  2. PIME 輸入法的選字框則是「橫列」的。

  3. 除了介面不太一樣，以我來說，習慣官方嘸蝦米輸入法的的字詞出字順序，使用 PIME 時，需要花一點時間習慣不太一樣的字詞出字順序。打個比方，「【」這個符號，我若用「,」+「[」這 2 個碼解字，那麼在官方嘸蝦米輸入法的字詞出字順序裡是排在第 3 位，而 PIME 的出字順序則是排在第 4 位。

  4. 使用同樣的 liu-uni.tab，卻有不一樣的字詞出字順序，其背後原因，我沒有去深入研究，也沒有去手動調整出字順序，單純就是花時間去習慣一下不同的字詞出字順序。

PIME 提供的設定選項

  1. 從工作列上的【中】點右鍵，可以開出 PIME 的選單。

  2. 再點擊選單裡的【設定輸入法模組】，就可以進入設定介面。

  3. 可以看到，設定的東西還滿多的，我只有進行截圖的動作，並沒有一一去試，有興趣的伙伴，可以自行摸索一番。

後記

系統及軟體版本訊息

  1. 本次經驗分享所使用的 Windows 是 Windows 11 家用版（24H2；OS 組建 26100.3476）。

  2. PIME 的版本是「PIME 輸入法 1.3.0-stable」。

  3. liu-uni.tab 的部分，取自「嘸蝦米輸入法 J 標準版 (x64) rev. 469」。

  4. 忘了自己是什麼時候開始用 PIME 的，也因為自己比較習慣官方的嘸蝦米輸入法的字詞出字順序，所以，停用 PIME 好一段時間。但，我認為 PIME 確實提供另一個輸入框架上的選擇，對我來說，使用起來相當穩定，入字不會有頓挫感。
