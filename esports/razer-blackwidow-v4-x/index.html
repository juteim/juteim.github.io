Razer BlackWidow V4 X（綠軸）的使用心得

發表於 2025-01-06 分類於 esports
文章字數： 1.2k 所需閱讀時間 ≈ 2 分鐘

前言

新的鍵盤是 Razer BlackWidow V4 X（綠軸），目前入手約 3 天，之所以會入手 Razer BlackWidow V4 X（綠軸）這支鍵盤，過程有一個小插曲。目前使用不到 1 星期，今天來簡單描述一下我的使用心得。

使用心得

  1. 低調的 Logo：RGB 的燈光一出，就沒有什麼所謂的低調，但，我都在家裡使用，就沒有什麼低不低調的問題，但，我也得說 Razer 的 Logo 真的不會那麼明顯。或是，是我沒有一直關注遊戲界的產品，才少見多怪。

  2. 手感：如果是拿 Razer Ornata V3 Tenkeyless 來比較的話，Razer BlackWidow V4 X 的回饋感及敲擊聲，更容易讓我上癮。

  3. 音量鍵：音量調整的按鍵，嚴格來說是一個寬度有點大的的滾筒（約 5 公分寬），滾動感滿不錯的。我在想，之所以把它做得那麼寬，應該是遊戲進行期間，盲操作時，容易觸碰與進行調整音量吧！？

  4. 巨集鍵：說真的，鍵盤左邊增加 6 顆巨集鍵，對我而言，並不好用，反而還滿容易打錯，有違我的習慣，所以，我一開始是把「M1」設定成「Ctrl」；「M6」設定成「Esc」，後來，索性就把 6 顆巨集鍵分別設定成隔壁的按鍵。即使如此，使用時我還是會提醒自己不要去打到巨集鍵，也因此，使用上就是覺得不順手。我相信巨集鍵，應該就是衝著遊戲而來，所以，如果入手這款鍵盤，主要是像我為了打字手感而來的話，可能還是要在當場試試展示用的鍵盤，確定自己能夠習慣，再決定要不要入手。

後記

  1. 「前言」提到的插曲，說起來是這樣的：我本來是從實體店面入手其他品牌的機械鍵盤（紅軸），那打起來是既順手又安靜一些些，但是，使用不到 7 天的日子裡，有一顆按鍵要不是一次出 2 個字，就是遲遲不出字，我只好趁著鑑賞期間，趕快跑一趟實體店面，經由銷售人員測試，證實我的感覺不是錯覺，確實有一顆按鍵有延遲的問題。

  2. 店家給出讓我換貨或是貼錢入手其他鍵盤的方案：我心裡面想著，似乎只有 Razer 家的產品，還沒有一入手就爆的狀況發生，所以，就快速地挑一支比入手不到 7 天就故障的鍵盤稍貴一點，同時是我能夠接受的價格，回到家一開始並有太多的問題，直到我要切換輸入法的時候，發現怎麼按了半天沒有反應，一度以為 Razer 也跟爆爆風潮，仔細一看，原來是我一直把巨集鍵「M1」當成是「Ctrl」鍵在使用。

  3. KEY FEEL：「回彈有勁、具敲擊感」這個說法，取自官網說明，入手後使用至今，也真如官網說法那般。只是，感受這件事，如果能夠當下體會，應該好過千言萬語的形容，我入手 Razer BlackWidow V4 X 的實體店面，有誠意地展示這一款鍵盤，只可惜隔著一層保鮮膜，我自己會覺得跟真實手感還是有一點點差距。

  4. 其他：由於我急著換到能夠正常使用的機械鍵盤，所以，沒有在當下花一點時間去測試巨集鍵的使用。另外，我還是喜歡紅軸多一點，未來再入手機械鍵盤，我會以紅軸為優先考量。
