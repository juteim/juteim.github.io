Razer Huntsman V3 Pro Tenkeyless 的使用心得

發表於 2025-01-19 分類於 esports
文章字數： 1k 所需閱讀時間 ≈ 2 分鐘

前言

Razer Huntsman V3 Pro Tenkeyless，在我使用約半個月後，決定也來寫一篇心得。

使用心得

  1. 質感 (1)鋁製頂板讓整個鍵盤的質感提升。 (2)右上角的巨集按鍵與旋鈕也都是鋁製，除了視覺上的高級感外，使用起來的感覺也不一樣。 (3)磁吸式硬質人造皮革護腕墊，兼顧質感與耐用性，遠遠超越 Razer Ornata V3 Tenkeyless 的磁吸式柔軟觸感護腕墊。

  2. 第 2 代 Razer 類比式光軸 (1)主打「快速觸發與可調整的觸發距離」，「「可調整的觸發距離」這個功能滿吸引我的，…儘管，我應該不太需要太細膩的設定。 (2)我剛開始用的是 Factory Default（也就是「出廠預設值」），總覺得輕輕點一下按鍵就會觸發效果，後來試著去使用其他的設定質，再回到 Factory Default 之後，覺得這樣的觸發距離最喜歡，並且也很快地就適應。

  3. 手感 (1)紋理表面的雙射出成型 PBT 鍵帽，有沒有差？對我而言，還真的有。 (2)只是，接手使用一段時間，我發現幾個按鍵中間有「毛髮」，一開始以為是我的毛髮掉在鍵盤上，後來才發現是按鍵底部的毛邊沒有處理掉，截至目前，只有 2 個按鍵有這個情形，實際上，也沒有影響到手感。

  4. 比 Razer BlackWidow V4 X（綠軸）安靜 (1)我使用這支鍵盤後，家人沒有主動表示我發出太大的躁音，所以，我認為應該是比較安靜的關係。 (2)至少，不會像 Razer BlackWidow V4 X（綠軸），讓我多打幾個字，就吵到家人而受到嫌棄。

  5. 逐漸適應 Tenkeyless (1)使用 Razer Ornata V3 Tenkeyless 半年期間，我還無法適應 Tenkeyless。 (2)接手 Razer Huntsman V3 Pro Tenkeyless 後，我試著再使用 2 個星期，想不到我會慢慢進入狀況。

後記

  1. 這支 Razer Huntsman V3 Pro Tenkeyless 鍵盤，是在 Razer Ornata V3 Tenkeyless 之後才購入。

  2. 謝謝家人拿 Razer Huntsman V3 Pro Tenkeyless 跟我的 Razer BlackWidow V4 X（綠軸）進行交換，家人跟我都感到滿開心的。

  3. 出乎意料之外，我已經能夠逐漸適應 Tenkeyless；從另外一個角度來看，也代表我未來很可能要多投入一些資源（錢）到鍵盤上。
