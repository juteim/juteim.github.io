Razer Basilisk V3 Pro

發表於 2025-03-03 分類於 esports
文章字數： 1.1k 所需閱讀時間 ≈ 2 分鐘

前言

相較於 Razer 家的鍵盤，我使用 Razer 家的滑鼠似乎更久遠一點。一開始我所使用的是 Razer Basilisk 有線滑鼠，由於使用起來非常舒服，在滑鼠上的橡膠開始「老化」後，我便決定入手無線版的 Razer Basilisk V3 X HyperSpeed。最終，才入手今天要提到的 Razer Basilisk V3 Pro。

使用心得

手感

  1. 延續我之前使用有線版本 Basilisk 的感受，也是我會決定入手無線版的主要原因。

  2. 也因為是無線，讓滑鼠自由自在一些，可能是因為這樣，讓我覺得手感有再提升一些些。

  3. 滑鼠本體上所使用的橡膠，一方面，容易沾灰塵；一方面，對於容易出手汗的伙伴，也不一定會喜歡。

續航力

  1. 我習慣讓滑鼠以 3,200 DPI 下去跑，發現電池的續航力，大概是 3 天左右，就需要充電。這一點，跟我過往使用他牌的無線滑鼠的使用經驗有差距。

  2. 入手之前，我知道這支滑鼠，主要功能是鎖定在電競使用，並沒有花時間去瞭解日常使用的話，續航力能不能符合我的習慣，因此，入手之後，遇到這樣的狀況。我想，就是自己沒有做足功課導致的。

搭配 Razer Mouse Dock Pro 的必要性

  1. 續航力與我的使用習慣有差距，一開始，我為了省錢，想說就直接插上 USB-C，秒變有線滑鼠就好。

  2. 想著想著，又會認為如果當成有線滑鼠，那何必多花錢買無線版本的 Basilisk。

  3. 所以，我便乾脆再花一點錢，入手 Razer Mouse Dock Pro，這樣的做法，確實有稍稍改善「我自己的習慣」與「續航力」之間衝突的問題。

終極形態

  1. 就像一開始的貼圖那般：最後的畫面就是 Razer Basilisk V3 Pro + Razer Mouse Dock Pro + Razer Basilisk V3 X HyperSpeed。。

  2. Razer Mouse Dock Pro 確實需要入手，然而，由於我的使用習慣，還是常常會讓使用中的 Basilisk 用到沒(電)力，所以，需要另外一隻 Basilisk 輪著用。

  3. 也就是說，就是只要使用中的 Basilisk 沒(電)力，就先下場休息，改讓另外一隻上場。

後記

  1. Basilisk 好像沒有出左手專用版(Left-handed)，所以，左撇子或習慣用左手操作滑鼠的伙伴，現階段應該跟 Basilisk 無緣。

  2. 說真的，以我的習慣來說，使用 Razer Basilisk V3 Pro 確實不太適合，我看了一下，即使是新版的 Razer Basilisk V3 Pro 35K，就算續航力來到 140 小時，也還是不符合我的習慣。

  3. 以我的使用需求而言，調高解析並習慣高解析後，就日常使用來說，我操作滑鼠進行移動的動作不需要太大，因此，我會認為有線版就夠我使用。
