使用 iTunes 備份 iTunes Store 上的已購音樂 feat. Windows 11

發表於 2025-05-10 分類於 apple
文章字數： 1.3k 所需閱讀時間 ≈ 2 分鐘

前言

iPhone 上有一些音樂，想要備份到電腦上。這件事，一直以來，我都使用官方的 App 搞定。今天這一則，紀錄的是 iTunes 進行備份的過程。

下載與備份程序

iPhone 或 iPad 端要做的事：開啟「同步資料庫」

  1. 使用「同步資料庫」，有一個前提就是需要訂閱 Apple Music 或 iTunes Match。

  2. 開啟「同步資料庫」的方法，官網有說明可供參考：透過訂閱 Apple Music 使用「同步資料庫」 - Apple 支援 (台灣)

iTunes 端要做的事

  1. 工欲善其事，必先有工具：iTunes，可以直接從 Microsoft Store 直接下載及安裝。

  2. 登入自己的 iTunes Store，並且授權給眼前的電腦。

  3. 從 iTunes App 的【編輯】→【偏好設定】進入「進階偏好設定」。

  4. 找到最右邊的【進階】，點進去把「保持 iTunes Media 資料夾的整理狀態」與「歌曲加入資料庫時，將其檔案複製到 iTunes Media 資料夾」都勾選起來。

  5. 回到 iTunes App 的主畫面，點一下畫面左手邊分格裡的【歌曲】後，再全選畫面右手邊分格內所有歌曲，，或是選擇想要下載的歌曲，再點滑鼠右鍵，從跳出的選單裡點【下載】。

  6. 可能會有一些音樂檔，下載時會出現錯誤訊息。

  7. 後續可以在 iTunes App 右上角，「搜尋工具列」右手邊，找到出現一個圈著驚嘆號的按鈕，點下去就可以把沒有下載到音樂檔，重新嘗試下載到電腦上（可能還是會有無法下載的檔案，但我個人的經驗是幾乎都能夠下載下來）。

Windows 端要做的事

  1. 最後，下載後的音樂檔，以 Windows 而言，檔案位置預設可能會在下列 2 個部徑底下（會有這樣的差別，主要是使用者有沒有另外用 OneDrive 備份到雲端導致）。

  2. 只要用關鍵字「.m4a」搜尋，就可以把檔案找出來。

後記

  1. iTunes 曾經是蘋果音樂服務的重要 App，然而，隨著時代的發展，iTunes 的功能逐漸被削弱，Apple Music 這支 App 出現後，iTunes 所剩的功能已經不多。

  2. 除了 Windows 上的 iTunes 可以辦到，老 Mac 仍在使用 iTunes 的機器，也同樣可行。也就以說，下載老 App：iTunes，Windows 的話，可以從 Microsoft Store 下載與安裝，沒有支援 Microsoft Store 的 Windows，可以跑一趟蘋果官網下載獨立的 iTunes 安裝程式；Mac 的話，要找老機器或是裝有舊一點的 macOS 來處理（例如，High Sierra 就可以）。

  3. 今天的步驟程序，原則上，應該使用 Apple Music 這支 App 也辦得到，但，我沒有試過，沒有百分百的把握。
